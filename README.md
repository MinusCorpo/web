# Minus Calendar Privacy Policy Site

このリポジトリは、Minus CalendarのプライバシーポリシーをGitHub Pagesで公開するためのものです。

## 使い方

1.  **プレビュー**:
    `index.html`をブラウザで開いて、デザインを確認してください。

2.  **内容の編集**:
    `index.html`内にある以下のプレースホルダー（`<span class="placeholder">...</span>`で囲まれている部分）を、ご自身の情報に書き換えてください。
    - `[あなたの氏名または組織名]`
    - `(公開URL)`
    - `[削除リクエスト用WebURL]`
    - `[あなたの連絡先メールアドレス]`
    - `[あなたの所在地...]`

3.  **GitHub Pagesでの公開**:
    - このリポジトリをGitHubにプッシュします。
    - GitHubのリポジトリ設定（Settings） > Pages に移動します。
    - Build and deployment の Source で「Deploy from a branch」を選択し、`main`ブランチの `/ (root)` を指定して保存します。
    - 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。
