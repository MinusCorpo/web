# プライバシーポリシー

このプライバシーポリシー（以下「本ポリシー」といいます）は、[あなたの氏名または組織名]（以下「当方」といいます）が提供するスマートフォン用アプリケーション「Minus Calendar」（以下「本アプリ」といいます）における、ユーザー情報の取り扱いを規定するものです。

本ポリシーは、(公開URL) にて公開されており、ユーザーはいつでも本アプリ内および公開Webページから最新の内容を確認することができます。

## 1. 同意の取得
本アプリは、情報の収集・利用を開始する前に、本ポリシーへの同意を取得します。初回起動時に同意画面を表示し、ユーザーの明示的な承認を得た場合にのみ利用を開始します。また、音声・画像データの収集については、該当機能の初回利用時に別途明示的な確認を行います。

## 2. 取得する情報
本アプリでは、以下の情報を取得し、ユーザー個人と結び付けて管理する場合があります。

### (1) ユーザーから直接提供される情報
*   **アカウント情報**: Googleサインイン等の認証時に取得される氏名、メールアドレス、プロフィール画像。
*   **カレンダーデータ**: ユーザーが本アプリに登録した予定、メモ、場所等の情報。
*   **音声・画像データ**: AIによる予定抽出機能を利用する際に入力された音声データや画像（写真）データ。

### (2) 外部サービスから取得する情報
*   **Googleカレンダー情報**: ユーザーの同意に基づき、Google Calendar APIを通じて取得するカレンダーの予定情報。
*   **購読・決済情報**: アプリ内課金（Proプラン等）の管理のために、RevenueCat、Apple（App Store）、Google（Google Play）から提供される決済ステータス情報。

### (3) 自動的に収集される情報
*   **利用状況・ログ情報**: 本アプリの利便性向上のため、Firebase Analytics等を通じて取得される、端末の種類、OSバージョン、アプリ内の操作ログ、クラッシュログ、広告識別子（IDFA/AAID）等。
*   **広告識別子の取り扱い**: 本アプリは広告表示を行わず、広告識別子（IDFA）はアプリ内の品質向上および利用状況分析のためにのみ使用します。第三者によるマーケティング目的の追跡（Tracking）には使用しません。

## 3. 情報の利用目的
取得した情報は、以下の目的で利用します。

*   本アプリの提供・維持、およびユーザー認証のため
*   カレンダーの同期および予定の管理機能の提供のため
*   AIを用いた音声・画像からの予定抽出機能の提供のため。なお、**AIによる解析（Gemini API等）に利用されるデータは機能提供の目的のみに使用され、基盤モデルの学習に利用されることはありません。**
*   アプリ内課金（Proプラン等）の適用・管理のため
*   ユーザーからのお問い合わせへの対応のため
*   不具合の修正や機能改善、マーケティング分析のため

## 4. 第三者への提供および外部送信
当方は、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません。

*   ユーザーの同意がある場合
*   法令に基づく場合
*   本アプリの運営に必要な範囲で、以下の信頼できるサービスプロバイダーに業務を委託する場合
    *   **Google Cloud / Firebase**: データ保存（Firestore）、認証（Firebase Auth）、AI解析（Gemini API）、解析（Analytics）。
    *   **RevenueCat**: サブスクリプション管理。
    *   **Apple Inc. / Google LLC**: アプリの配信および決済処理。

## 5. Google API データの取扱い
本アプリは、Google API（カレンダー等）を通じて取得した情報を、本アプリ内での表示・編集、およびユーザー自身のGoogleカレンダーへの同期以外に使用することはありません。
本アプリは **Google API Services User Data Policy** を遵守します。特に、Google APIから受信した情報を他アプリへ転送する場合、**限定公開（Limited Use）要件を含むポリシー**を遵守し、取得したデータを第三者広告目的で利用・販売することはありません。

## 6. データの保存とセキュリティ
当方は、ユーザー情報の漏洩、滅失または毀損の防止その他の安全管理のために、適切な措置を講じます。
*   **保存場所**: Firebase等のセキュアなインフラストラクチャを利用します。
*   **国外への移転**: 本アプリで取得したデータは、サービス提供元（Google等）が管理する国外のサーバーに保存される場合があります。
*   **セキュリティ**: 通信の暗号化（SSL/TLS）およびデータベースのアクセス制限を徹底します。

## 7. データの保持期間と削除
*   **カレンダー・アカウントデータ**: ユーザーがアカウント削除をリクエストするまで保存されます。削除リクエスト後、30日以内に当方のサーバーから完全に消去されます。
*   **解析ログデータ**: 統計および改善目的で最大90日間保持された後、匿名化または削除されます。
*   **削除方法**: 
    1.  **アプリ内**: 設定画面の「アカウント削除」からいつでも即時に実行可能です。
    2.  **Webサイト**: アプリをアンインストール済み等の理由でアプリ内操作が困難な場合は、[削除リクエスト用WebURL] のフォームより削除をリクエストいただけます。リクエスト受領後、速やかに対応いたします。

## 8. ユーザーの権利
ユーザーは当方に対し、以下の権利を行使することができます。
*   **アクセス権**: 自身の個人情報の開示を求める権利。
*   **修正・削除権**: 不正確な情報の修正や、情報の削除を求める権利。
*   **同意の撤回**: 収集に対する同意をいつでも撤回できる権利（アンインストールまたはアカウント削除により行使）。
これらの要求については、下記のお問い合わせ先にて受け付けます。
